SS Playlist Generator (Web Edition)
Version: v1.0
Built by: ShortStroke & Solace

Description:
A simple web tool that lets users enter a vibe or list of songs, then generates a themed YouTube playlist.
The current version uses a fixed placeholder link for testing and teaching purposes.

Features:
- Custom vibe input
- Animated loading message (jet ski inspired)
- Playlist link output
- Fully local (no account or API needed yet)

Next Planned Features:
- Real-time playlist generation using GPT suggestions
- GPT integration for auto-list creation
- Visual polish + branding enhancements
- Option to save playlist to YouTube (future API-based version)

For: Ellie and the vibe keepers of the world
